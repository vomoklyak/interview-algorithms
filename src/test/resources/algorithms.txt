Задача о рюкзаке 0-1
Имеется n предметов, p(i) > 0 ценность, w(i) > 0 вес.
Нужно найти такую комбинацию предметов чтобы суммарная ценность предметов была максимальна при заданном суммарном весе
рюкзака W.
Задача решается с помощью рекурентных соотношений.
Считаем что задача решена для k первых предметов и нужно добавить k+1.
Возможны два варианта.
Первый мы помещаем k+1 предмет и тогда ценность рюкзака будет ценность p(k+1) плюс максимальная ценность рюкзака для предыдущих k предметов и оставшейся вместительности
рюкзака W-w(k+1).
Второй мы не помещаем k+1 предмет и тогда ценность рюкзака совпадает с предыдущим шагом.
Проходя по всем предметам и выбирая максимальную ценность из двух вариантов выше получим максимальную ценность для рюкзака с заданной массой из всех предметов.
